# Mixed language station names in English app version

## Environment
- Device: Android Samsung Galaxy A52s 5G
- OS Version: 14
- App Version: 3.3.9
- Language: English
- Network: WiFi

## Severity
Medium

## Priority
Medium

## Description
Some station names are displayed in Arabic while the application language is set to English.

## Preconditions
- App language is set to English

## Steps to Reproduce
1. Open the application.
2. Start booking a trip.
3. Select city as Marsa Matrouh.
4. View available stations.

## Actual Result
Some stations appear in Arabic while others appear in English.

## Expected Result
All station names should be displayed in English.
